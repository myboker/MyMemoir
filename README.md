# MyMemoir / Life Nebula

一个面向个人自传写作的网页应用。用户可以记录人生某一阶段的经历，系统会将经历整理成事件索引、关系星云图和章节化自传草稿。

## 功能

- 记录人生事件，而不是按天写日记。
- 动态星云图展示事件之间的关系。
- 事件重要性会隐性影响节点大小和呼吸速度。
- 搜索事件、事件索引、代理整理信息默认折叠，保持界面简洁。
- 点击“梳理传记”后，系统会整理事件、识别自然时间、建立关系、规划章节并生成自传草稿。
- 支持按不同风格生成自传文本。
- 支持修改记录、回退、导出全文和导出章节图片。

## 直接打开

可以直接双击 `index.html` 使用本地降级版整理逻辑。

## 启用强模型服务

如需启用强模型传记代理，请先设置环境变量，再启动本地服务：

```powershell
$env:BLTCY_API_KEY="你的 API Key"
node server.js
```

然后打开：

```text
http://localhost:8787
```

也可以继续用 `file://` 打开页面，前端会尝试调用：

```text
http://localhost:8787/api/organize
```

## 隐私说明

- 不要把 API Key 写入源码。
- `.env`、`api.md`、临时响应文件已经加入 `.gitignore`。
- 自传内容默认保存在浏览器本地存储中。
